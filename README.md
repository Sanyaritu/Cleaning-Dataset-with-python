# Amazon Sales Performance Analysis

## Overview
This project analyzes e-commerce sales data to uncover revenue trends and product performance insights. 
The goal is to transform raw sales data into actionable business insights.

## Dataset
- Source: Kaggle (Amazon Sales Report)
- Data includes order details, product categories, order status, quantities, and revenue.

## Objectives
- Clean and structure raw sales data
- Analyze monthly revenue trends
- Identify top revenue-generating products
- Provide business recommendations

## Tools Used
- Python (pandas, matplotlib)
- Microsoft Excel

## Key Insights
- Revenue showed a noticeable decline in June 2022

## Business Recommendations
- Focus inventory and marketing efforts on top-performing products

- Optimize or reconsider low-performing product categories
